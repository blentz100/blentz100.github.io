Welcome to Bootstrap Bootcamp

This website is a project for the Promineo Tech Front End Software Developer program. I'll be sharing some examples of how to use bootstrap to style your webpage.


Alerts

Alerts give contextual feedback to users based on a variety of conditions. Here is what the code looks like: 

<div class="alert alert-primary" role="alert">
	Primary Alert
</div>
<div class="alert alert-secondary" role="alert">
	Secondary Alert
</div>
<div class="alert alert-success" role="alert">
	Success Alert
</div>
<div class="alert alert-danger" role="alert">
	Danger Alert
</div>
<div class="alert alert-warning" role="alert">
	Warning Alert
</div>
<div class="alert alert-info" role="alert">
	Info Alert
</div>
<div class="alert alert-light" role="alert">
	Light Alert
</div>
<div class="alert alert-dark" role="alert">
	Dark Alert
</div>

Output

Primary Alert
Secondary Alert
Success Alert
Danger Alert
Warning Alert
Info Alert
Light Alert
Dark Alert

Alert Links

Alert links provide convenient way to color links in alerts with a matching color to the alert background color - using the alert-link class.

<div class="alert alert-primary" role="alert">
	Primary Alert with a <a href="https://www.google.com"
	class="alert-link" target="_blank"> colored
		link</a>
</div>
<div class="alert alert-primary" role="alert">
	Primary Alert with a <a href="https://www.google.com"
		target="_blank"> regular
		link</a>
</div>
<div class="alert alert-secondary" role="alert">
	Secondary Alert with a <a href="https://www.google.com"
	class="alert-link" target="_blank"> colored
		link</a>
</div>
<div class="alert alert-secondary" role="alert">
	Secondary Alert with a <a href="https://www.google.com"
		target="_blank">
		regular link</a>
</div>

Output

Primary Alert with a colored link
Primary Alert with a regular link
Secondary Alert with a colored link
Secondary Alert with a regular link
