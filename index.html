HTML Basics

HTML, or Hypertext Markup Language, is a way to format content to be viewable over a web browser. On this site I'll share some information for beginner users of HTML.

In HTML there are two types of elements when it comes to content that the browsers will display: Block and Inline. Block level elements are rectangular in shape and span the entire width of their partent element. Multiple inline elements can share the same horizontal space. Below are few block level elements and inline level elements with examples. These are the two kinds of elements that can appear within the document body. There are other elements that can appear within the document head that we will not cover here.

Block Level Elements

  • <address> ... </address>

    This element is used to format the address of the document author. My address is
    805 N 4th Ave, Phoenix, AZ 85003.

  • <blockquote> ... </blockquote>

    This element is used to format block level quotes. Use this element if your quote is longer than a few lines and you want the browser to indent the whole quote relative to the rest of the text. Here is an example:
    How does one cope with evil? Not by fighting it but by understanding. In understanding, it disappears. How does one cope with darkness? Not with one’s fist. You don’t chase darkness out of a room with a broom. You turn on a light. ~ Awareness by Anthony De Mello

  • <del> ... </del>

    This element is used to format text so there is a line going through it. It can used as block level element or an inline level element. Example: The lazy quick brown fox jumped over the lazy dog.

Inline Elements

  • <q> ... </q>

    This element is used to format an inline level quote as opposed to a block level quote. Example: The Eat real food. Mostly plants. Not too much. ~Michael Pollan

  • <audio> ... </audio>

    Adds playable audio to a page. Click play below to hear an audio recording of a Western Screech Owl courtesy of the Arizona Field Ornithologists Sound Library ( www.azfo.org).
